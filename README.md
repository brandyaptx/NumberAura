# NumberAura - Free Numerology Tools

A lightweight Next.js tool site for US search traffic around low-competition numerology calculator keywords.

## Included pages

- Home page
- Name Numerology Calculator
- Address Numerology Calculator
- Wedding Date Numerology Calculator
- Personal Year Numerology Calculator
- Life Path Number Calculator
- Word Numerology Calculator

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Static SEO-friendly pages
- No database
- Vercel-ready

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Import this GitHub repository into Vercel.
2. Framework preset: Next.js.
3. Build command: `npm run build`.
4. Output directory: leave default.
5. Add your custom domain later in Vercel project settings.
